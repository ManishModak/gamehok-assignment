# Gamehok Frontend Assignment

A high-fidelity, responsive gaming platform UI built with **Next.js 15**, **TypeScript**, and **Tailwind CSS**. This project was developed as part of the frontend developer recruitment process for **Gamehok**.

## 🚀 Live Demo
**URL:** [https://gamehok-assignment-zeta.vercel.app](https://gamehok-assignment-zeta.vercel.app)

## ✨ Key Features
- **Pixel-Perfect Implementation:** Translated Figma designs into clean, reusable React components with a focus on visual accuracy.
- **Responsive Design:** Optimized for both mobile (Bouncy Bird UI focus) and desktop layouts.
- **Modern Tech Stack:** Leveraging the latest Next.js App Router features for performance and SEO.
- **Dynamic Assets:** Integrated SVG icons and optimized game imagery for a premium gaming feel.
- **Interactive Components:** Features horizontal carousels, custom tournament rounds schedule, and a sticky mobile navigation system.

## 🛠️ Tech Stack
- **Framework:** [Next.js 15+](https://nextjs.org/) (App Router)
- **Language:** [TypeScript](https://www.typescriptlang.org/)
- **Styling:** [Tailwind CSS 4.0](https://tailwindcss.com/)
- **Components:** [Radix UI / Base UI](https://base-ui.com/) (for accessible primitives)
- **Icons:** [Lucide React](https://lucide.dev/) & Custom SVGs
- **Deployment:** [Vercel](https://vercel.com/)

## 📦 Getting Started

### Prerequisites
- Node.js 18.x or later
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/ManishModak/gamehok-assignment.git
   ```
2. Navigate to the project directory:
   ```bash
   cd gamehok-assignment
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Run the development server:
   ```bash
   npm run dev
   ```
5. Open [http://localhost:3000](http://localhost:3000) in your browser.

## 📁 Project Structure
- `src/app`: Next.js App Router pages and layouts.
- `src/components`: Reusable UI components (Layout, Home, Tournament).
- `src/lib`: Mock data and utility functions.
- `public`: Static assets including optimized images and SVGs.
